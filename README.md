# 金销路 - 企业官网托管平台

这是一个使用Python和Flask框架开发的企业官网托管平台。

## 开发环境

- Python
- Flask

## 生产环境

- Nginx
- Gunicorn

## 使用方法

1. 克隆这个仓库到你的本地机器上。
2. 安装所有的依赖包。你可以使用以下命令进行安装：

```bash
pip install -r requirements.txt
```
3. 配置环境变量

复制 .env.example 文件并重命名为 .env。
打开 .env 文件并填入你的环境变量值。

4. 运行服务。你可以使用以下命令进行运行：

```bash
python index.py
```
